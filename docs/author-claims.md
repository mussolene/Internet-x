# Author Claims

## Safe Claims

- Internet-X is an identity-first overlay architecture.
- The repository contains a runnable reference implementation with authenticated locator update.
- The repository makes a strong case for Internet-X as a good design point under explicit overlay, mobility, and inspectability constraints.

## Unsafe Claims

- Internet-X is the first system to separate identity from locator.
- Internet-X proves a better Internet architecture in the universal sense.
- The current implementation is post-quantum secure.
- The current repository is ready for operational deployment.
